# Scala 集合

Scala 集合分为可变集合（mutable）和不可变集合（immutable），可变集合在初始化之后可以自动进行扩展，就是意味着可变集合允许添加、删除、修改元素；不可变集合在初始化之后不允许增删改元素；相关类位于 `scala.collection` 包下，其中 `scala.collection.mutable` 和 `scala.collection.immutable` 分别为可变集合和可变集合的包。
